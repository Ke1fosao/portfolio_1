# About Scroll Story Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a unique scroll-driven About page that tells Дмитро Ковтунович's personal and professional story and ends with a clear collaboration CTA.

**Architecture:** The page remains a server component for metadata and semantic content. A focused client controller observes chapters, updates the active story index, and writes scroll progress to a CSS variable. All visuals and responsive behavior live in a prefixed global stylesheet with no new dependencies.

**Tech Stack:** Next.js App Router, React 19, TypeScript, CSS, IntersectionObserver.

## Global Constraints

- Preserve the existing header, theme system and footer.
- Use only confirmed biography details from the approved design.
- Do not add third-party animation libraries.
- Respect `prefers-reduced-motion`.
- Keep mobile content linear and fully readable without animation.

**Files:**
- Create: `components/about/about-story-controller.tsx`

**Interfaces:**
- Consumes: DOM elements with `[data-about-chapter]` and the root element with `[data-about-root]`.
- Produces: `data-active-chapter` on the root, `.is-visible` on chapters, and CSS variable `--about-scroll` in the range `0` to `1`.
