# Portfolio — Дмитро Ковтунович

Сучасне персональне портфоліо full-stack розробника, створене на **Next.js** і **TypeScript**.

Дизайн натхнений стриманою естетикою Magic Portfolio, але реалізація, компоненти, стилі та контент написані з нуля для комерційного використання без копіювання ліцензійно обмеженого коду.

## Основні можливості

- головна сторінка з позиціонуванням і вибраними роботами;
- окрема сторінка всіх проєктів;
- детальна сторінка для кожного кейсу;
- сторінка «Про мене»;
- світла й темна тема;
- адаптивний дизайн;
- metadata, Open Graph, sitemap і robots;
- проєкти зі статусами «Завершено», «У процесі» та «Концепт».

## Стек

- Next.js 16;
- React 19;
- TypeScript;
- CSS без сторонньої UI-бібліотеки.

## Запуск

```bash
npm install
npm run dev
```

Відкрийте `http://localhost:3000`.

## Перед деплоєм

Замініть `https://portfolio.example.com` у:

- `app/layout.tsx`;
- `app/sitemap.ts`;
- `app/robots.ts`.

Після цього виконайте:

```bash
npm run typecheck
npm run build
```
