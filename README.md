# Portfolio Website

Односторінковий персональний сайт-портфоліо для презентації навичок, досвіду, послуг і реалізованих проєктів.

## Про проєкт

Сайт буде розроблений як сучасний, швидкий та адаптивний вебзастосунок. Основна мета — професійно представити розробника, показати його роботи та надати потенційним клієнтам зручний спосіб зв’язку.

## Запланований стек

- **Next.js** — основний React-фреймворк;
- **TypeScript** — типізація та надійніша розробка;
- **React** — побудова компонентів інтерфейсу;
- **Tailwind CSS** — стилізація та адаптивний дизайн;
- **Framer Motion** — плавні анімації та переходи;
- **Vercel** — розгортання готового сайту.

## Заплановані секції

- Головний екран;
- Про мене;
- Навички та технології;
- Послуги;
- Проєкти та кейси;
- Досвід і освіта;
- Контактна форма;
- Посилання на соціальні мережі.

## Основні вимоги

- односторінкова структура;
- адаптація під мобільні пристрої, планшети та комп’ютери;
- сучасний і професійний дизайн;
- плавна навігація між секціями;
- оптимізація швидкості завантаження;
- базова SEO-оптимізація;
- зручна та доступна контактна форма.

## Поточний статус

Проєкт знаходиться на початковому етапі. Наразі створено репозиторій і базову документацію. Наступним кроком буде ініціалізація Next.js-проєкту та розробка структури головної сторінки.
